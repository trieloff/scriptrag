# Installation

## Requirements

- Python 3.11 or higher
- SQLite 3.38 or higher (for vector support)

## Install from PyPI

```bash
pip install scriptrag
```

## Install from Source

```bash
git clone https://github.com/trieloff/scriptrag.git
cd scriptrag
pip install -e .
```

## Development Installation

```bash
git clone https://github.com/trieloff/scriptrag.git
cd scriptrag
make setup-dev
```

## Verify Installation

```bash
scriptrag --version
```
